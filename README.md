## Client

See [rtd-client/ directory](rtd-client/).

You can run the app in a browser with:
```bash
ionic serve
```

Or you can emulate ios with:
```bash
ionic build ios
ionic emulate ios
```

Or develop live on real iOS device:

```bash
ionic run ios --livereload --consolelogs --serverlogs
```
